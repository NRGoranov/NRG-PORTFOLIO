# Restaurant Website Design Style

## Design Philosophy

### Color Palette
- **Primary**: Deep charcoal (#2C2C2C) for sophisticated elegance
- **Secondary**: Warm cream (#F8F6F0) for inviting warmth  
- **Accent**: Rich burgundy (#8B2635) for luxury and appetite appeal
- **Supporting**: Soft gold (#D4AF37) for premium highlights

### Typography
- **Display Font**: Playfair Display - Elegant serif for headings and hero text
- **Body Font**: Inter - Clean, modern sans-serif for readability
- **Accent Font**: Cormorant Garamond - Sophisticated serif for menu items

### Visual Language
- **Sophisticated Minimalism**: Clean layouts with purposeful white space
- **Culinary Artistry**: High-quality food photography as hero elements
- **Warm Hospitality**: Inviting color tones and organic textures
- **Premium Quality**: Subtle luxury details without ostentation

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth scroll animations and element transitions
- **Splide.js**: Elegant image carousels and gallery sliders
- **ECharts.js**: Data visualization for restaurant statistics
- **p5.js**: Creative background effects and interactive elements
- **Pixi.js**: Advanced visual effects for hero sections

### Effect Implementation
- **Hero Background**: Subtle particle system with warm golden tones
- **Text Animations**: Staggered letter reveals with elegant timing
- **Image Transitions**: Ken Burns effect on hero imagery
- **Scroll Motion**: Gentle parallax with 8% maximum translation
- **Hover States**: 3D tilt effects on menu cards and buttons

### Header Effect
- **Aurora Gradient Flow**: Warm gradient animation in burgundy and gold
- **Floating Elements**: Subtle geometric shapes for visual interest
- **Depth Layers**: Multiple background layers for rich dimensionality

### Styling Approach
- **Consistent Background**: Warm cream base throughout all sections
- **Visual Hierarchy**: Clear distinction between content areas
- **Mobile-First**: Responsive design optimized for all devices
- **Performance**: Optimized loading with progressive image enhancement
